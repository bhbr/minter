# minter
INTERactive Math objects

**Minter** is a pen-based user-interface prototype designed in the spirit of other recent work (Chalktalk, Magic Paper). No menus, no modes, just you and your pen on an open canvas building and connecting objects!

The underlying API (written in TypeScript) is still in early development. It allows for easy extension, as well as the integration of existing mathematical web content such as Cindy applets.

## Requirements

- Python 3
- TypeScript
- rollup (via npm)

## Usage

- run build/build.sh
- open minter/minter.html on a virtual server
